# 宏勤 出貨明細轉單系統 (v3.3)

這是一個基於 HTML/JavaScript 的輕量化網頁工具，專為宏勤生活科技設計。可將 Excel 或 CSV 格式的訂單資料，快速轉換成格式統一、適合列印與存檔的 PDF 出貨明細單。

## 🌟 v3.3 最新功能與特色
1. **新增「訂單時間」顯示與修復**：
   - 自動擷取訂單時間的「年月日」，並顯示於 PDF 的訂單編號旁。
   - 內建日期格式轉換器，自動辨識並修復 Excel 內部的日期序號（例如將 `46155` 自動轉回 `2026-05-13`），確保時間顯示正確。
2. **全單客製化編輯功能**：
   - 突破以往只能修改備註的限制，現在連同 **出貨日期** 以及 **每一筆商品明細（SKU、品名、數量）** 皆可手動修改。
   - 修改後該筆訂單將會被系統「記憶並鎖定」，優先顯示手動輸入的內容，不受上方 Excel 欄位變動的影響。
3. **一鍵重設與防呆機制**：
   - 單筆訂單提供「重設此單」按鈕，可隨時刪除手動修改紀錄，恢復與 Excel 資料的連動。
   - 總控台新增「清除所有手動修改」按鈕，方便在載入新檔案或重新對應欄位卡住時，一次性解除所有訂單的鎖定。
4. **支援拖曳客製欄位**：可無限新增額外資訊欄位，並支援拖曳 (⠿) 調整顯示順序。

## 🚀 使用步驟
1. **載入資料**：將 Excel (`.xlsx`, `.xls`) 或 CSV 檔案拖曳或點擊上傳至灰色虛線框內。
2. **欄位對應**：系統會自動嘗試對應「訂單編號」、「訂單時間」、「SKU」、「商品名稱」、「數量」與「備註」。若對應錯誤，可透過下拉選單手動修正。
3. **客製額外資訊 (選用)**：點擊「新增欄位」可增加需要在單據上顯示的額外資訊（可指定讀取 Excel 欄位或手動輸入固定值）。
4. **產出預覽與單筆修改**：
   - 點擊「1. 產出預覽」，系統會在下方生成所有單據的預覽畫面。
   - 若某張單據需要微調，點擊該單據左側的「📝 修改此單內容」即可展開面板編輯，修改後請點擊儲存。
5. **下載 PDF**：確認預覽無誤後，點擊「2. 下載所有 PDF」，系統會將所有單據合併為一個 PDF 檔案並自動下載。

## ⚠️ 注意事項
- 本工具完全在瀏覽器端運行（Offline-first），您的客戶與訂單資料**不會**被上傳到任何外部伺服器，確保資安無虞。
- 建議使用 Google Chrome 或 Microsoft Edge 瀏覽器以獲得最佳的 PDF 輸出排版效果。
