# 宏勤 出貨明細轉單系統

將電商平台下載的訂單數據 (Excel/CSV) 自動轉換為標準 A4 PDF 出貨單，支援分組、分頁與自動頁碼。

## 💡 使用說明
1. **上傳檔案**：將 Excel 直接拖入網頁。
2. **欄位確認**：檢查對應的原始欄位是否正確。
3. **生成預覽**：點擊按鈕確認排版效果。
4. **下載 PDF**：獲取完整出貨清單，每張單據右下角皆有自動生成的頁碼。

## 🛠 技術說明
- 純前端處理，資料不經過伺服器，保障公司商務 privacy。
- 使用技術：Vanilla JS, SheetJS, html2pdf.js。
